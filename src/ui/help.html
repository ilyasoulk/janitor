SkyShade Help

Getting Started

SkyShade is a text anonymization tool that helps you quickly remove or replace personally identifiable information in text. Here's how to use it:

  1. Anonymize text on a webpage: Select text on any webpage, click the SkyShade icon, and click "Anonymize Selection".
  2. Anonymize your own text: Open the extension, paste text in the input box, and click "Anonymize Text".
  3. Copy results: Click the "Copy Result" button to copy the anonymized text to your clipboard.

What Gets Anonymized

SkyShade identifies and replaces:

  • Names of people
  • Locations and places
  • Organizations
  • Email addresses
  • Other personally identifiable information

Tips & Tricks

Keyboard Shortcut: Use Ctrl+Shift+Y to quickly open SkyShade.

Right-Click Menu: You can also right-click on selected text and choose "Anonymize Selection" from the context menu.

For Large Texts: For very large texts, consider anonymizing in smaller chunks for better performance.

Troubleshooting

Text isn't being anonymized properly

If certain information isn't being anonymized:

  • Try selecting a larger context (more surrounding text)
  • Ensure the text is in a commonly recognized format
  • Some specialized terminology may not be recognized as PII

How to undo an anonymization

If you've anonymized text and need to revert to the original content:

  • Simply refresh the page to restore the original text
  • Anonymization happens locally in your browser and doesn't permanently modify the website

Extension isn't working on a page

Some websites have restrictions that prevent extensions from accessing their content. In these cases:

  • Copy the text and use the "Anonymize Text Input" method instead
  • Try refreshing the page

Website Compatibility
